# 核心功能

**MediaPipe4U** 主要通过动画蓝图中的节点来实现动作捕捉，所以动画蓝图节点就是 **MediaPipe4U** 的核心功能。   

**MediaPipe4U** 包含以下动画蓝图节点，本章将详细介绍：

- **MediaPipe Pose Node**: 姿态节点
- **MediaPipe Hand Node**: 手部节点
- **MediaPipe Location Node**: 位置（位移）节点
- **MediaPipe Head Node**: 头部节点
