# MediaPipe4U 简介

**MediaPipe4U** 是一个 UnrealEngine 插件，将 Google MediaPipe 技术带入 Unreal Engine ，可以通过普通摄像头、视频文件、图片等媒体，
实时的驱动虚幻引擎种的 3D 角色（数字化身）。 


# 功能

- 全身/半身动作捕捉
- 手指捕捉
- 面容表情捕捉
- 角色移动
- 动画数据导出（BVH）
- 通过摄像头
- 从摄像头捕获
- 从视频（mp4）捕获
- 从图片（jpg,png）捕获
- 关节扭曲矫正
- 动画平滑
- Ground IK: 让脚可以贴合地面
- Nvidia AR: 通过 Nvidia 机器 AI 技术，获得媲美苹果 Arkit 效果面部 BlendShape 算解
