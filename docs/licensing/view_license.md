
# 如何查看授权有效期

可以打开项目设置窗口，选中左侧 **插件**->**MediaPipe4U** ，细节面板中将会看到插件和授权信息。

[![License](./images/plugin_settings.jpg "License")](./images/plugin_settings.jpg)   

# 运行时获取授权有效期

你可以在运行时通过 MediaPipeUtils::PluginInfo 函数获取插件的授权有效期。
